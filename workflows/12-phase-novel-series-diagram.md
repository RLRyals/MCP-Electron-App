## Key Workflow Features

### ğŸ”„ Two Major Loops

1. **Book Loop** (Phases 3-11)
   - Repeats for each of 5 books
   - Each iteration produces one complete book

2. **Chapter Loop** (Phase 5)
   - Nested inside book loop
   - Writes 25-30 chapters per book

### ğŸšª Two Quality Gates

1. **NPE Validation Gate** (Phase 4)
   - Ensures solid planning before writing
   - Loops back to Phase 3 if score < 80

2. **Final Quality Gate** (Phase 9)
   - Ensures publication readiness
   - Loops back to Phase 7 if readiness < 90

### âœ… Six Approval Points

Authors review and approve at:
1. Phase 2: Series architecture
2. Phase 3: Book outline (per book)
3. Phase 4: NPE validation results
4. Phase 7: Developmental edit suggestions
5. Phase 8: Line edit changes
6. Phase 9: Final quality check
7. Phase 12: Series completion package

### â±ï¸ Estimated Timeline

**Per Book (90,000 words):**
- Planning & Validation: 2-4 hours
- Chapter Writing: 6-8 hours (1-2 min/chapter)
- Editing & QA: 2-3 hours
- **Total: ~10-15 hours**

**Complete 5-Book Series (450,000 words):**
- **Total: ~50-75 hours**

*Actual time varies based on LLM speed, review time, and revisions needed*
